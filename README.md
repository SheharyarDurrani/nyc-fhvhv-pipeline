# NYC FHVHV Rideshare Pipeline 🚕

## Overview
End-to-end data engineering pipeline analysing **20.9 million** NYC rideshare trips (Jan 2026).

## Key Findings
- Uber holds **73%** market share vs Lyft's 27%
- Peak demand is at **6-7pm** on weekdays
- 54,956 bad rows cleaned from raw data

## Tech Stack
- Python, Pandas, DuckDB
- Jupyter Notebooks
- Git & GitHub

## Data Source
[NYC TLC Trip Record Data](https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page)

## How to Run
1. Download dataset from NYC TLC website
2. Place in project root folder
3. Run `notebooks/01_exploration.ipynb`
